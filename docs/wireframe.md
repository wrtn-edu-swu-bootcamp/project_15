### 1-1. 급수 선택 호버 툴팁

각 급수 카드에 마우스를 올리면 해당 급수의 언어 능력을 간략히 보여줍니다.

**호버 시 툴팁 내용** (CEFR 기준, 50자 이내):

| 급수 | 툴팁 내용 |
|------|----------|
| A1 (입문) | 기본 인사, 자기소개 등 간단한 일상 표현 가능 |
| A2 (초급) | 일상적 정보 교환, 간단한 묘사와 대화 가능 |
| B1 (중급1) | 익숙한 주제에 대해 의견을 표현하고 대처 가능 |
| B2 (중급2) | 복잡한 텍스트 이해, 원어민과 유창한 대화 가능 |
| C1 (고급1) | 학술적, 전문적 상황에서 유창하게 소통 가능 |
| C2 (고급2) | 원어민에 준하는 완벽한 언어 구사력 보유 |

```
* 호버 예시 (A1 카드에 마우스 올린 경우):

     ┌─────┐
     │ A1  │  ← 호버 상태 (테두리 강조)
     │입문 │
     └─────┘
        │
        ▼
   ┌────────────────────────────────────┐
   │ 기본 인사, 자기소개 등              │
   │ 간단한 일상 표현 가능               │
   └────────────────────────────────────┘
```
