# 프랑스어 기사 기반 학습 서비스 기획안 (MVP)

---

## 1. 문제 정의

### 1.1 문제 상황

불어불문학과 학생이 졸업요건을 채우기 위해 DELF 시험을 준비하던 중, "DELF 공부는 신문으로 하는 게 좋다"는 조언을 받았다. 그러나 실제로 프랑스어 신문 기사를 읽어보니 **어느 부분을 짚어서 공부해야 학습에 도움이 될지 막막했다**.

### 1.2 핵심 문제

| 문제 | 설명 |
|------|------|
| **급수별 학습 자료 부족** | DELF/DALF 급수(A1~C2)에 맞는 실제 기사 기반 학습 자료를 찾기 어려움 |
| **어휘 급수 파악 어려움** | 기사를 읽을 때 어떤 단어가 어느 급수인지 알 수 없어 학습 우선순위를 정하기 어려움 |
| **문맥 속 문법 이해 부족** | 문법책의 예문과 실제 기사의 문장이 달라 문법 규칙을 실제 텍스트에 적용하기 어려움 |
| **학습 연속성 부재** | 기사를 읽고 나서 복습하거나 점검할 방법이 없어 학습 효과가 떨어짐 |

### 1.3 DELF/DALF 시험의 특성

DELF/DALF는 프랑스 교육부 인증 공인 자격증으로, **실제 프랑스어 자료(기사, 방송, 문서)를 활용**하여 청취, 독해, 작문, 구술 4개 영역을 종합 평가한다. 따라서 실제 기사로 학습하는 것이 시험 준비에 효과적이지만, 급수별로 어떤 부분을 집중해야 할지 가이드가 필요한 상황이다.

---

## 2. 서비스 개요

### 2.1 서비스 소개

**서비스명**: LangSavor (랭세이버)  
**브랜드 컨셉**: Language(언어) + Savor(음미하다)

> 미식가가 요리를 음미하듯, 기사 속 언어를 천천히 맛보며 학습합니다.

LangSavor는 프랑스어 기사 텍스트를 분석하여 CEFR 급수(A1~C2)별로 단어, 표현, 문법을 자동 분류하고, 시각적으로 표시하여 학습을 돕는 서비스입니다.

### 2.2 핵심 가치 제안

- **급수 맞춤형 분석**: DELF/DALF 시험 급수에 맞춰 어휘와 문법을 자동 분류
- **즉시 이해**: 기사를 읽으면서 모르는 단어와 문법을 바로 확인
- **능동적 학습**: 퀴즈를 통해 학습 내용을 즉시 복습하고 점검

---

## 3. 솔루션

### 3.1 해결 방법

1. **기사 텍스트 분석**: LLM(Gemini API)을 활용하여 텍스트 내 단어, 표현, 문법을 자동 분석
2. **급수별 자동 분류**: 각 언어 요소에 CEFR 급수(A1~C2) 태깅
3. **시각적 하이라이팅**: 급수별 색상으로 기사 본문에 하이라이팅 표시
4. **인터랙티브 학습**: 호버 시 즉시 뜻/설명 확인, 더블클릭으로 단어장 저장
5. **퀴즈를 통한 복습**: 학습한 기사 기반 빈칸 채우기 퀴즈로 학습 효과 검증

### 3.2 작동 방식

```
텍스트 입력 → LLM 분석 → 급수 태깅 → 하이라이팅 렌더링 → 인터랙션
```

## 4. 핵심 기능

### 4.1 텍스트 붙여넣기

사용자가 프랑스어 기사 텍스트를 직접 복사하여 붙여넣는 방식. 저작권 이슈를 최소화하고 사용자가 관심 있는 주제를 자유롭게 선택할 수 있음.

**프랑스 언론사 가이드**: 급수별 추천 언론사 소개
- A1-A2: 1jour1actu (무료)
- B1-B2: France 24, RFI (무료)
- B2-C2: Le Monde, Le Figaro, Libération (부분 유료)

### 4.2 언어 분석

| 분석 항목 | 내용 |
|----------|------|
| **단어** | 품사, 성/수, 원형, 한국어/프랑스어 뜻, 급수 분류 |
| **표현** | 관용구, 숙어, 연어(collocation), 급수 분류 |
| **문법** | 시제, 법, 구문 구조, 급수별 문법 포인트 분류 |

### 4.3 하이라이팅 시스템

**급수별 색상 체계**:
- A1: 노랑 | A2: 주황 | B1: 초록 | B2: 파랑 | C1: 빨강 | C2: 보라

**이중 표시**:
1. **기사 본문 하이라이팅**: 급수별 색상으로 표시, 호버 시 팝업으로 상세 정보
2. **별도 목록**: 사이드바에 단어/표현/문법 탭으로 정리, 급수 필터링 가능

**사용자 추가 기능**: 텍스트 선택으로 학습하고 싶은 단어/표현 추가 (점선 테두리로 구분)

### 4.4 퀴즈

학습한 기사 기반 빈칸 채우기 퀴즈
- **유형**: 객관식 / 주관식
- **특징**: 실제 기사 문장 활용, 급수별 난이도 조절, 즉시 채점 및 해설

### 4.5 사용자 관리

- **비회원**: 세션 기반 임시 사용, 브라우저 종료 시 데이터 소멸
- **회원**: Google 로그인, 학습 기록/단어장/퀴즈 결과 영구 저장, 학습 통계 확인

## 5. 사용자 플로우

### 5.1 기본 플로우

```
1. 서비스 접속
   ↓
2. 학습 급수 선택 (A1~C2)
   ↓
3. 프랑스어 기사 텍스트 붙여넣기
   ↓
4. AI 분석 및 하이라이팅 표시
   ↓
5. 기사 읽기 + 호버로 단어/문법 확인
   ↓
6. (회원) 원하는 단어 더블클릭하여 냉장고(단어장) 저장
   ↓
7. 퀴즈 풀기
   ↓
8. (회원) 마이페이지에서 학습 이력 확인
```

### 5.2 주요 사용 시나리오

**시나리오: DELF B1 준비생**
1. B1 급수로 설정
2. RFI에서 문화 기사 텍스트 복사하여 붙여넣기
3. B1 급수 단어(초록색)에 집중하며 기사 읽기
4. 모르는 단어는 더블클릭하여 냉장고에 저장
5. 퀴즈로 학습 내용 점검
6. 다음 날 냉장고에서 저장한 단어 복습

## 6. MVP 범위

### 6.1 포함 기능

| 기능 | 설명 |
|------|------|
| 텍스트 붙여넣기 | 사용자가 직접 기사 텍스트 복사하여 입력 |
| 언어 분석 | LLM(Gemini)을 활용한 급수별 단어/표현/문법 자동 분류 |
| 하이라이팅 | 본문 하이라이팅 + 사이드바 목록 |
| 품사별 분류 | 명사/동사/형용사/부사/기타 품사별 정리 |
| 퀴즈 | 빈칸 채우기 (객관식/주관식) |
| 사용자 계정 | Google 로그인, 데이터 저장 |
| 언론사 가이드 | 급수별 추천 언론사 소개 |
| 이용약관 | 저작권 관련 책임 고지 |

### 6.2 제외 기능 (향후 추가)

- 발음 듣기 (TTS)
- 작문 연습
- 말하기 연습
- AI 챗봇
- 커뮤니티
- 모바일 앱
