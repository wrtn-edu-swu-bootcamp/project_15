# 프랑스어 기사 기반 학습 서비스 기획안 (MVP)

---

## 1. 서비스 개요

### 1.1 서비스 소개

**서비스명**: (추후 확정)

프랑스어를 학습하는 한국인을 위한 기사 기반 언어 학습 서비스입니다. 실제 프랑스어 기사를 활용하여 CEFR(유럽공통언어표준) 급수에 맞는 어휘, 표현, 문법을 체계적으로 학습할 수 있도록 돕습니다.

### 1.2 핵심 가치 제안

- **실용적인 학습**: 교과서가 아닌 실제 프랑스어 기사로 살아있는 언어를 학습
- **급수 맞춤형 분석**: DELF/DALF 시험 급수(A1~C2)에 맞춰 어휘와 문법을 분류
- **즉시 이해**: 기사를 읽으면서 모르는 단어와 문법을 바로 확인
- **능동적 학습**: 퀴즈를 통해 학습 내용을 즉시 복습하고 점검

### 1.3 목표 사용자

**주 타겟**: 한국인 프랑스어 학습자

- DELF/DALF 시험을 준비하는 학습자
- 프랑스어 읽기 능력을 향상시키고 싶은 학습자
- 실제 프랑스어 콘텐츠로 학습하고 싶은 학습자
- 대학교 불어불문학과 학생
- 프랑스 유학/취업을 준비하는 사람

**사용자 특성**:
- 기초~고급 수준의 프랑스어 학습자
- 자격증 취득을 목표로 체계적인 학습을 원함
- 실제 프랑스어 텍스트를 읽을 때 어휘/문법 장벽을 느낌

### 1.4 핵심 문제 정의

| 문제 | 상세 설명 |
|------|----------|
| 급수별 학습 자료 부족 | DELF/DALF 급수에 맞는 실제 기사 기반 학습 자료를 찾기 어려움 |
| 어휘 급수 파악 어려움 | 기사를 읽을 때 어떤 단어가 어느 급수인지 알기 어려움 |
| 문맥 속 문법 이해 부족 | 문법책의 예문과 실제 기사의 문장이 달라 적용이 어려움 |
| 학습 연속성 부재 | 기사를 읽고 나서 복습하거나 점검할 방법이 없음 |

### 1.5 솔루션

기사 텍스트를 분석하여 급수별로 단어, 표현, 문법을 자동 분류하고, 이를 시각적으로 표시하여 학습을 돕습니다. 또한 학습한 내용을 바탕으로 퀴즈를 제공하여 학습 효과를 높입니다.

## 2. 핵심 기능

### 2.1 기사 제공 시스템

#### URL 입력 크롤링
- 사용자가 원하는 프랑스어 기사 URL을 입력하면 텍스트를 자동 추출
- 다양한 프랑스어 뉴스 사이트 지원 (Le Monde, Le Figaro, France 24 등)

#### 큐레이션 기사 라이브러리
- 서비스에서 미리 선별한 양질의 기사 제공
- 급수별, 카테고리별로 분류하여 제공

**카테고리 구성**:
| 카테고리 | 설명 | 예시 |
|----------|------|------|
| 시사/뉴스 | 최신 뉴스 및 시사 이슈 | 정치, 경제, 사회 |
| 문화/예술 | 프랑스 문화, 예술, 영화, 음악 | 칸 영화제, 루브르 전시 |
| 스포츠 | 스포츠 뉴스 | 축구, 올림픽, 테니스 |
| 과학/기술 | 과학 발견, 기술 트렌드 | 환경, 우주, IT |
| 라이프스타일 | 일상, 요리, 여행 | 프랑스 생활, 관광지 |

### 2.2 언어 분석 기능

#### 단어 분석
- 기사 내 단어를 CEFR 급수(A1~C2)별로 자동 분류
- 단어의 품사, 성/수, 원형 제공
- 한국어 뜻과 프랑스어 정의 병기

#### 표현 분석
- 관용구, 숙어, 연어(collocation) 인식
- 급수별 표현 분류
- 실제 사용 맥락과 함께 설명

#### 문법 분석
- 문장 내 문법 구조 파악 (시제, 법, 구문 등)
- 급수별 문법 포인트 분류
- 상세한 문법 규칙 설명 + 예문
- 해당 기사 문맥에서의 활용법 설명

### 2.3 학습 자료 제공 방식

#### 이중 표시 시스템

**1. 기사 본문 하이라이팅**
- 급수별 색상 구분으로 단어/표현 하이라이팅
- 마우스 호버 시 즉시 설명 팝업 표시

**급수별 색상 체계**:
| 급수 | 색상 | 의미 |
|------|------|------|
| A1 | 초록 | 입문 수준 |
| A2 | 파랑 | 초급 수준 |
| B1 | 노랑 | 중급 1 수준 |
| B2 | 주황 | 중급 2 수준 |
| C1 | 빨강 | 고급 1 수준 |
| C2 | 보라 | 고급 2 수준 |

**2. 별도 목록 제공**
- 사이드바 또는 하단에 정리된 목록 표시
- 탭으로 단어/표현/문법 구분
- 급수별 필터링 가능

#### 다국어 설명
- 모든 설명은 한국어와 프랑스어 병기
- 학습자가 원하는 언어로 우선 표시 설정 가능

### 2.4 퀴즈 및 학습 기능

#### 빈칸 채우기 퀴즈

학습한 기사를 바탕으로 자동 생성되는 퀴즈 제공

**퀴즈 유형**:

| 유형 | 설명 | 예시 |
|------|------|------|
| 객관식 빈칸 | 4-5개 선택지에서 정답 선택 | "Il ___ à Paris." (a) va (b) vais (c) vont (d) allez |
| 주관식 빈칸 | 직접 단어 입력 | "Elle ___ française." → est |

**퀴즈 특징**:
- 학습한 기사의 실제 문장을 활용
- 급수별 난이도 조절
- 즉시 채점 및 정답/해설 제공
- 오답 시 관련 문법/어휘 설명 표시

### 2.5 사용자 관리

#### 비회원 모드
- 회원가입 없이 모든 기본 기능 이용 가능
- 세션 기반으로 임시 데이터 유지
- 브라우저 종료 시 데이터 소멸

#### 회원 모드
- 이메일 또는 소셜 로그인으로 가입
- 제공 기능:
  - 학습 기록 영구 저장
  - 개인 단어장에 단어/표현 저장
  - 퀴즈 결과 및 점수 추적
  - 학습 통계 확인

## 3. 사용자 플로우

### 3.1 비회원 사용자 플로우

```
1. 서비스 접속
   ↓
2. 학습 급수 선택 (A1~C2)
   ↓
3. 기사 선택
   - 선택한 급수에 맞는 큐레이션 목록에서 선택
   - 또는 URL 직접 입력
   ↓
4. 기사 학습
   - 하이라이팅된 기사 읽기
   - 단어/표현/문법 목록 확인
   - 호버하여 즉시 설명 확인
   ↓
5. 퀴즈 풀기
   - 빈칸 채우기 문제 풀이
   - 즉시 채점 및 해설 확인
   ↓
6. 학습 종료 (데이터 미저장)
```

### 3.2 회원 사용자 플로우

```
1. 로그인
   ↓
2. 학습 급수 선택 (A1~C2)
   ↓
3. 기사 선택
   - 선택한 급수에 맞는 큐레이션 목록에서 선택
   - 또는 URL 직접 입력
   ↓
4. 기사 학습
   - 하이라이팅된 기사 읽기
   - 원하는 단어/표현 단어장에 저장
   ↓
5. 퀴즈 풀기
   - 문제 풀이 및 채점
   - 결과 자동 저장
   ↓
6. 마이페이지에서 학습 이력 확인
   - 저장한 단어장 복습
   - 퀴즈 성적 통계 확인
```

### 3.3 주요 사용 시나리오

#### 시나리오 1: DELF B1 준비생
> 김민지(25세)는 DELF B1 시험을 3개월 앞두고 있다. 문법은 어느 정도 공부했지만 실제 프랑스어 기사를 읽을 때 모르는 단어가 많아 막막하다.

1. 서비스에 접속하여 B1 급수로 설정
2. B1 급수에 맞게 필터링된 큐레이션 기사 중 관심 있는 문화 기사 선택
3. 기사를 읽으면서 B1 급수 단어(노란색)에 집중
4. 모르는 단어는 단어장에 저장
5. 퀴즈로 학습 내용 점검
6. 다음 날 단어장에서 저장한 단어 복습

#### 시나리오 2: 프랑스어 초보자
> 박준호(30세)는 프랑스 여행을 앞두고 프랑스어 기초를 배우고 있다. 쉬운 기사부터 읽어보고 싶다.

1. 서비스에 접속 (비회원)
2. A1-A2 급수로 설정
3. 해당 급수에 맞는 라이프스타일 카테고리의 쉬운 기사 선택
4. 초록색(A1), 파란색(A2) 단어 위주로 학습
5. 간단한 퀴즈로 기초 단어 점검

## 5. 급수별 학습 기준

### 5.1 CEFR 급수 체계

| 급수 | CEFR 레벨 | 설명 |
|------|-----------|------|
| A1 | 입문 | 기초적인 일상 표현 이해 |
| A2 | 초급 | 간단한 일상 대화 가능 |
| B1 | 중급 1 | 일상적인 주제에 대해 의사소통 |
| B2 | 중급 2 | 복잡한 텍스트 이해, 유창한 대화 |
| C1 | 고급 1 | 다양한 주제에 대해 유창하게 표현 |
| C2 | 고급 2 | 원어민에 준하는 수준 |

### 5.2 급수별 어휘 기준

| 급수 | 어휘 수준 | 주요 어휘 유형 |
|------|----------|---------------|
| A1 | 기초 일상 단어 | 인사, 숫자, 가족, 음식, 색상, 기본 동사 |
| A2 | 일상 표현 확장 | 일상 활동, 장소, 쇼핑, 교통, 날씨 |
| B1 | 추상적 개념 시작 | 감정, 의견, 직업, 건강, 미디어 |
| B2 | 전문 용어 일부 | 시사, 환경, 경제 기초, 문화 관련 어휘 |
| C1 | 폭넓은 전문 용어 | 학술, 비즈니스, 정치, 과학 전문 용어 |
| C2 | 원어민 수준 | 관용어, 속담, 문학적 표현, 전문 분야 심화 |

### 5.3 급수별 문법 기준

| 급수 | 주요 문법 포인트 |
|------|-----------------|
| A1 | 현재형 (être, avoir, -er 동사), 정관사/부정관사, 형용사 성수 일치, 기본 부정문, 의문문 |
| A2 | 복합과거, 반과거, 근접미래, 대명동사, 목적어 대명사, 비교급/최상급 |
| B1 | 접속법 현재 (기초), 단순미래, 조건법 현재, 관계대명사 (qui, que, où), 수동태 |
| B2 | 접속법 활용 심화, 과거분사 일치, 현재분사/제롱디프, 복합 관계대명사, 간접화법 |
| C1 | 단순과거, 전과거, 조건법 과거, 접속법 과거, 문어체 표현 |
| C2 | 문어체 고급 시제, 접속법 반과거/대과거, 고급 구문, 문체적 도치 |

## 8. 향후 확장 가능성

### 8.1 단기 확장 (MVP 이후 3-6개월)

- **레벨 테스트**: 서비스 시작 시 간단한 테스트로 사용자 급수 자동 판정
- **발음 듣기**: TTS 연동으로 단어/문장 발음 재생
- **학습 알림**: 복습 주기에 맞춰 푸시 알림 발송

### 8.2 중기 확장 (6-12개월)

- **개인화 추천**: 학습 패턴 분석 기반 맞춤 기사 추천
- **작문 연습**: 학습한 단어/표현으로 문장 작성 연습
- **소셜 기능**: 친구 추가, 학습 경쟁, 리더보드

### 8.3 장기 확장 (12개월 이후)

- **다언어 확장**: 스페인어, 독일어, 이탈리아어 등 CEFR 기반 언어
- **모바일 앱**: iOS/Android 네이티브 앱 출시
- **프리미엄 기능**: 고급 분석, 무제한 저장, 광고 제거
- **AI 튜터**: 1:1 대화형 학습 지원

## 부록: 용어 정의

|------|------|
