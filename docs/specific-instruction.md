# 프랑스어 기사 기반 학습 서비스 기획안 (MVP)

---

## 1. 서비스 개요

### 1.1 서비스 소개

**서비스명**: LangSavor (랭세이버)

**브랜드 컨셉**: Language(언어) + Savor(음미하다)

> 미식가가 요리를 음미하듯, 기사 속 언어를 천천히 맛보며 학습합니다.

LangSavor는 프랑스어를 학습하는 한국인을 위한 기사 기반 언어 학습 서비스입니다. 실제 프랑스어 기사를 활용하여 CEFR(유럽공통언어표준) 급수에 맞는 어휘, 표현, 문법을 체계적으로 학습할 수 있도록 돕습니다.

마치 좋은 요리의 풍미를 음미하듯, 기사라는 풍부한 재료에서 언어의 맛(단어, 표현, 문법)을 하나하나 느끼고 자신의 것으로 만들어 가는 학습 경험을 제공합니다.

### 1.2 핵심 가치 제안

- **실용적인 학습**: 교과서가 아닌 실제 프랑스어 기사로 살아있는 언어를 학습
- **급수 맞춤형 분석**: DELF/DALF 시험 급수(A1~C2)에 맞춰 어휘와 문법을 분류
- **즉시 이해**: 기사를 읽으면서 모르는 단어와 문법을 바로 확인
- **능동적 학습**: 퀴즈를 통해 학습 내용을 즉시 복습하고 점검

### 1.3 목표 사용자

**주 타겟**: 한국인 프랑스어 학습자

- DELF/DALF 시험을 준비하는 학습자
- 프랑스어 읽기 능력을 향상시키고 싶은 학습자
- 실제 프랑스어 콘텐츠로 학습하고 싶은 학습자
- 대학교 불어불문학과 학생
- 프랑스 유학/취업을 준비하는 사람

**사용자 특성**:
- 기초~고급 수준의 프랑스어 학습자
- 자격증 취득을 목표로 체계적인 학습을 원함
- 실제 프랑스어 텍스트를 읽을 때 어휘/문법 장벽을 느낌

### 1.4 DELF/DALF 시험의 특성

DELF/DALF는 프랑스 교육부가 인증하는 공인 자격증으로, 실질적인 언어 구사 능력을 종합적으로 평가합니다.

**시험 구성**:
| 영역 | 평가 내용 |
|------|----------|
| 청취 (Compréhension orale) | 실제 방송, 대화 등을 듣고 이해하는 능력 |
| 독해 (Compréhension écrite) | 기사, 문서 등 실제 텍스트를 읽고 이해하는 능력 |
| 작문 (Production écrite) | 주어진 주제에 대해 논리적으로 글을 작성하는 능력 |
| 구술/인터뷰 (Production orale) | 면접관과의 대화를 통해 말하기 능력 평가 |

**시험의 핵심 특징**:
| 특성 | 설명 |
|------|------|
| 실질적 언어 능력 평가 | 단순 문법 암기가 아닌, 실제 상황에서의 언어 활용 능력을 평가 |
| 다양한 주제 영역 | 일상 대화부터 정치, 사회, 경제, 예술 등 복잡한 주제까지 포괄 |
| 실제 자료 기반 출제 | 프랑스어권에서 실제 사용되는 기사, 방송, 문서를 활용 |
| 과락 제도 | 각 영역별 최소 점수 기준이 있어, 4개 영역 모두 균형 잡힌 실력 필요 |

> **핵심 포인트**: DELF/DALF는 어느 한 영역만 잘해서는 합격할 수 없습니다. 청취, 독해, 작문, 구술 4개 영역 모두에서 해당 급수가 요구하는 기본적인 실력을 갖춰야 하며, 이를 위해서는 다양한 주제의 실제 프랑스어 콘텐츠에 균형 있게 노출되는 것이 중요합니다.

### 1.5 기존 학습 방법의 한계

| 한계점 | 상세 설명 |
|--------|----------|
| 교과서 중심 학습 | 정형화된 예문 위주로, 실제 사용되는 다양한 표현에 노출 부족 |
| 주제 범위 불일치 | 일상 회화 위주 학습으로, 시사/사회/경제/예술 등 시험 출제 주제와 괴리 |
| 실제 자료 접근성 부족 | 급수에 맞는 실제 프랑스어 콘텐츠를 찾고 분석하기 어려움 |
| 영역별 불균형 | 독해나 문법에 치중하여 청취, 작문, 구술 등 다른 영역 대비 미흡 |
| 단편적 학습 | 단어, 문법, 독해가 분리되어 통합적 언어 능력 향상에 한계 |

### 1.6 본 서비스의 차별점

**고급 급수 (B2~C2) 학습자에게**:

기존 앱들은 일상 회화 중심으로, 고급 급수 시험이 요구하는 복잡한 주제와 고도화된 언어 표현을 다루지 못합니다. 본 서비스는 이러한 한계를 해결합니다.

| 차별점 | 설명 |
|--------|------|
| 고도화된 언어 표현 학습 | 기사에서 사용되는 격식체, 문어체, 고급 구문 등 실제 고급 표현 습득 |
| 맥락 파악 능력 향상 | 긴 텍스트 속에서 논지를 파악하고 뉘앙스를 이해하는 훈련 |
| 전문 분야 용어 학습 | 정치, 경제, 사회, 과학, 예술 등 각 분야의 전문 어휘 체계적 습득 |
| 시험 출제 경향 일치 | 실제 시험에서 활용되는 기사/문서 유형과 동일한 자료로 학습 |

**초급 급수 (A1~B1) 학습자에게**:

초급 급수는 기존 앱으로도 준비할 수 있지만, 본 서비스는 장기적인 학습 관점에서 차별화된 가치를 제공합니다.

| 차별점 | 설명 |
|--------|------|
| 실제 프랑스어 조기 노출 | 처음부터 살아있는 프랑스어에 익숙해져, 고급 급수로의 전환이 자연스러움 |
| 문맥 속 학습 | 단어와 문법을 실제 맥락에서 학습하여 장기 기억에 효과적 |
| 상위 급수 로드맵 | 현재 급수 학습 중에도 상위 급수 어휘/표현을 미리 접하며 학습 방향 설정 |
| 흥미 기반 학습 | 관심 있는 주제의 기사로 학습하여 동기 부여 및 학습 지속성 향상 |
| 교과서 너머의 표현 | 일상에서 실제로 쓰이지만 교과서에 없는 표현들을 자연스럽게 습득 |

### 1.7 솔루션

LangSavor는 기사 텍스트를 분석하여 급수별로 단어, 표현, 문법을 자동 분류하고, 이를 시각적으로 표시하여 학습을 돕습니다. 

마치 요리사가 재료를 손질하듯, 기사라는 원재료에서 학습에 필요한 언어 요소를 추출하고 급수별로 정리하여 제공합니다. 학습자는 자신의 수준에 맞는 언어의 맛을 음미하며, 퀴즈를 통해 학습 내용을 자신의 것으로 소화합니다.

### 1.8 핵심 문제 정의

| 문제 | 상세 설명 |
|------|----------|
| 급수별 학습 자료 부족 | DELF/DALF 급수에 맞는 실제 기사 기반 학습 자료를 찾기 어려움 |
| 어휘 급수 파악 어려움 | 기사를 읽을 때 어떤 단어가 어느 급수인지 알기 어려움 |
| 문맥 속 문법 이해 부족 | 문법책의 예문과 실제 기사의 문장이 달라 적용이 어려움 |
| 학습 연속성 부재 | 기사를 읽고 나서 복습하거나 점검할 방법이 없음 |

## 2. 핵심 기능

### 2.1 기사 입력 시스템

#### 텍스트 붙여넣기 (핵심 기능)

사용자가 원하는 프랑스어 기사를 직접 복사하여 붙여넣는 방식으로, 서비스의 **핵심 입력 방식**입니다.

| 특징 | 설명 |
|------|------|
| 자유로운 소스 선택 | 사용자가 관심 있는 주제의 기사를 직접 선택 |
| 즉시 분석 | 붙여넣기 후 바로 AI 분석 시작 |
| 저작권 투명성 | 사용자가 소스를 직접 관리하여 저작권 이슈 최소화 |
| 맞춤형 학습 | 관심 분야, 난이도를 사용자가 직접 조절 |

**사용 흐름**:
1. 프랑스 언론사 가이드에서 급수에 맞는 언론사 확인
2. 해당 언론사에서 관심 있는 기사 선택
3. 기사 텍스트 복사
4. 서비스에 붙여넣기
5. AI 분석 시작

#### 프랑스 언론사 가이드

사용자가 기사를 가져올 수 있는 프랑스 주요 언론사를 소개합니다.

**종합 일간지 (B2-C2)**:
| 언론사 | 특징 | 접근성 |
|--------|------|--------|
| Le Monde | 프랑스 대표 일간지, 심층 분석 기사 | 부분 유료 |
| Le Figaro | 보수 성향 종합 일간지 | 부분 유료 |
| Libération | 진보 성향 일간지 | 부분 유료 |

**뉴스 채널 (B1-B2)**:
| 언론사 | 특징 | 접근성 |
|--------|------|--------|
| France 24 | 국제 뉴스, 명확한 문체 | 무료 |
| RFI | 프랑스어 학습자용 콘텐츠 제공 | 무료 |

**학습자 친화적 매체 (A1-B1)**:

프랑스 대표 교육 출판사(Play Bac Presse, Milan Presse, Bayard Presse)에서 발행하는 매체로, 쉬운 어휘를 사용하면서도 정확한 문법과 신뢰할 수 있는 정보를 제공합니다.

| 언론사 | 대상 | 추천 급수 | 접근성 |
|--------|------|----------|--------|
| Le Petit Quotidien | 6-10세 | A1 | 유료 구독 |
| 1jour1actu | 8-13세 | A1-A2 | 무료 |
| Phosphore | 15-18세 | B1-B2 | 유료 구독 |

> **Tip**: 초급 학습자는 1jour1actu에서 가벼운 맛보기로 시작하여, 점차 Le Monde의 깊은 풍미에 도전해 보세요!

### 2.2 언어 분석 기능

#### 단어 분석
- 기사 내 단어를 CEFR 급수(A1~C2)별로 자동 분류
- 단어의 품사, 성/수, 원형 제공
- 한국어 뜻과 프랑스어 정의 병기

#### 표현 분석
- 관용구, 숙어, 연어(collocation) 인식
- 급수별 표현 분류
- 실제 사용 맥락과 함께 설명

#### 문법 분석
- 문장 내 문법 구조 파악 (시제, 법, 구문 등)
- 급수별 문법 포인트 분류
- 상세한 문법 규칙 설명 + 예문
- 해당 기사 문맥에서의 활용법 설명

### 2.3 학습 자료 제공 방식

#### 이중 표시 시스템

**1. 기사 본문 하이라이팅**
- 급수별 색상 구분으로 단어/표현 하이라이팅
- 마우스 호버 시 즉시 설명 팝업 표시

**급수별 색상 체계**:
| 급수 | 색상 | 의미 |
|------|------|------|
| A1 | 노랑 | 입문 수준 |
| A2 | 주황 | 초급 수준 |
| B1 | 초록 | 중급 1 수준 |
| B2 | 파랑 | 중급 2 수준 |
| C1 | 빨강 | 고급 1 수준 |
| C2 | 보라 | 고급 2 수준 |

**하이라이팅 작동 방식**:

1. 텍스트 입력 - 사용자가 기사 텍스트 붙여넣기
2. LLM 분석 - Gemini API로 텍스트 전송하여 언어 요소 분석
3. 급수 태깅 - 각 단어/표현/문법에 CEFR 급수(A1-C2) 부여
4. 렌더링 - 급수별 색상을 적용하여 기사 본문에 하이라이팅 적용
5. 인터랙션 - 호버 시 팝업으로 상세 정보 표시

| 단계 | 설명 |
|------|------|
| 텍스트 입력 | 사용자가 입력한 기사 원문을 서버로 전송 |
| LLM 분석 | LLM이 단어, 표현, 문법 구조를 파악하고 급수 판정 |
| 급수 태깅 | 분석 결과를 구조화된 데이터(JSON)로 반환 |
| 렌더링 | 프론트엔드에서 급수별 색상을 적용하여 화면에 표시 |
| 인터랙션 | 하이라이팅된 요소 호버 시 뜻, 품사, 예문 등 팝업 |

**사용자 하이라이팅 추가 기능**:

사용자가 직접 텍스트를 선택하여 학습하고 싶은 단어/표현을 추가할 수 있습니다.

| 항목 | 설명 |
|------|------|
| 활성화 | "재료 손질하기" 버튼 클릭으로 토글 (ON/OFF 상태 시각적 구분) |
| 추가 방식 | 활성화 상태에서 텍스트 드래그로 선택 |
| 급수 판정 | 선택한 텍스트를 LLM이 자동으로 분석하여 급수 부여 |
| 저장 | 냉장고(단어장)에 자동 저장 |

**하이라이팅 구분**:
| 유형 | 표시 방식 |
|------|----------|
| 자동 분석 | 급수별 색상 (노랑~보라) 배경 |
| 사용자 추가 | 급수별 색상 + 점선 테두리로 구분 |

**2. 별도 목록 제공**
- 사이드바 또는 하단에 정리된 목록 표시
- 탭으로 단어/표현/문법 구분
- 급수별 필터링 가능

#### 다국어 설명
- 모든 설명은 한국어와 프랑스어 병기
- 학습자가 원하는 언어로 우선 표시 설정 가능

### 2.4 퀴즈 및 학습 기능

#### 빈칸 채우기 퀴즈

학습한 기사를 바탕으로 자동 생성되는 퀴즈 제공

**퀴즈 유형**:

| 유형 | 설명 | 예시 |
|------|------|------|
| 객관식 빈칸 | 4-5개 선택지에서 정답 선택 | "Il ___ à Paris." (a) va (b) vais (c) vont (d) allez |
| 주관식 빈칸 | 직접 단어 입력 | "Elle ___ française." → est |

**퀴즈 특징**:
- 학습한 기사의 실제 문장을 활용
- 급수별 난이도 조절
- 즉시 채점 및 정답/해설 제공
- 오답 시 관련 문법/어휘 설명 표시

### 2.5 사용자 관리

#### 비회원 모드
- 회원가입 없이 모든 기본 기능 이용 가능
- 세션 기반으로 임시 데이터 유지
- 브라우저 종료 시 데이터 소멸

#### 회원 모드
- 이메일 또는 소셜 로그인으로 가입
- 제공 기능:
  - 학습 기록 영구 저장
  - 개인 단어장에 단어/표현 저장
  - 퀴즈 결과 및 점수 추적
  - 학습 통계 확인

### 2.6 이용약관 및 저작권 안내

서비스 이용 시 다음 사항에 대해 사용자 동의를 필수로 받습니다.

**저작권 관련 동의 사항**:
| 항목 | 내용 |
|------|------|
| 비영리 사용 | 본 서비스에 입력하는 텍스트는 개인 학습 목적으로만 사용해야 하며, 영리적 목적의 사용을 금지합니다 |
| 저작권 책임 | 사용자가 입력하는 텍스트의 저작권 관련 법적 책임은 사용자 본인에게 있습니다 |
| 면책 조항 | 서비스 제공자는 사용자가 입력한 콘텐츠로 인한 저작권 침해에 대해 책임지지 않습니다 |

**동의 절차**:
- 서비스 최초 이용 시 이용약관 동의 필수
- 텍스트 붙여넣기 시 저작권 관련 주의사항 안내 문구 표시

### 3.3 주요 사용 시나리오

#### 시나리오 1: DELF B1 준비생 - 어휘 확장
> 김민지(25세)는 DELF B1 시험을 3개월 앞두고 있다. 문법은 어느 정도 공부했지만 실제 프랑스어 기사를 읽을 때 모르는 단어가 많아 막막하다.

1. 서비스에 접속하여 B1 급수로 설정
2. 언론사 가이드에서 B1 추천 매체(France 24, RFI)를 확인
3. RFI에서 관심 있는 문화 기사를 찾아 텍스트 복사
4. 서비스에 텍스트 붙여넣기 후 분석 시작
5. 기사를 읽으면서 B1 급수 단어(초록색)에 집중
6. 모르는 단어는 냉장고(단어장)에 저장
7. 퀴즈로 학습 내용 점검
8. 다음 날 냉장고에서 저장한 단어 복습

#### 시나리오 2: 문법 지루함 탈출 - 맥락 속 문법 학습
> 이수현(22세)은 대학교에서 프랑스어를 전공하고 있다. 접속법, 관계대명사 등 문법책으로 공부하면 규칙은 이해되지만, 실제 문장에서 어떻게 쓰이는지 감이 안 잡히고 금방 잊어버린다.

1. 서비스에 접속하여 B2 급수로 설정
2. 언론사 가이드에서 B2 추천 매체(Le Monde, Le Figaro)를 확인
3. Le Monde에서 관심 있는 경제 기사를 찾아 텍스트 복사
4. 서비스에 텍스트 붙여넣기 후 분석 시작
5. 기사 본문에서 하이라이팅된 문법 요소 확인
   - "Il faut que le gouvernement **prenne** des mesures..." → 접속법 현재 (prendre)
   - "L'entreprise **qui a été fondée** en 2010..." → 관계대명사 + 수동태
6. 문법 탭에서 해당 문법의 상세 설명과 기사 내 활용 예시 확인
7. 같은 문법이 다른 문장에서 어떻게 변형되는지 비교 학습
8. 빈칸 퀴즈로 동사 활용, 관계대명사 선택 등을 직접 연습
9. 문맥 속에서 반복 노출되어 자연스럽게 문법 감각 체득

> **학습 효과**: 문법책의 추상적인 규칙이 실제 기사에서 동사, 명사, 형용사와 어떻게 결합하는지 직접 보면서 학습하여, 지루한 암기 대신 맥락적 이해로 장기 기억에 효과적

#### 시나리오 3: 프랑스어 초보자 - 실제 프랑스어 첫 경험
> 박준호(30세)는 프랑스 여행을 앞두고 프랑스어 기초를 배우고 있다. 쉬운 기사부터 읽어보고 싶다.

1. 서비스에 접속 (비회원)
2. A1-A2 급수로 설정
3. 언론사 가이드에서 A1-A2 추천 매체(1jour1actu)를 확인
4. 1jour1actu에서 쉬운 라이프스타일 기사를 찾아 텍스트 복사
5. 서비스에 텍스트 붙여넣기 후 분석 시작
6. 노란색(A1), 주황색(A2) 단어 위주로 학습
7. 간단한 퀴즈로 기초 단어 점검

## 4. 화면 구성

### 4.1 홈 페이지

**구성 요소**:
- 서비스 소개 영역
  - 핵심 가치 제안 문구 ("기사 속 언어를 음미하며 프랑스어를 마스터하세요")
  - 주요 기능 간략 소개 (신선한 재료, 급수별 손질, 실력 테스트)
- 급수 선택 영역
  - 6개 급수 카드 (A1~C2)
  - 각 카드 호버 시 급수별 설명 툴팁 표시
  - 급수 선택 후 "학습 시작하기" 버튼 → 텍스트 붙여넣기 페이지로 이동
- 로그인/회원가입 버튼 (헤더)
- 언론사 가이드 바로가기 링크

### 4.2 언론사 가이드 페이지

**구성 요소**:
- 급수별 추천 언론사 목록
  - A1-B1: 학습자 친화적 매체 (1jour1actu 등)
  - B1-B2: 뉴스 채널 (France 24, RFI)
  - B2-C2: 종합 일간지 (Le Monde, Le Figaro, Libération)

  - 특징 설명
  - 추천 급수

  - 웹사이트 링크
- 사용법 안내
  - 기사 찾는 방법
  - 텍스트 복사 방법
  - 서비스에 붙여넣기 방법

## 8. 향후 확장 가능성

### 8.1 단기 확장 (MVP 이후 3-6개월)

- **레벨 테스트**: 서비스 시작 시 간단한 테스트로 사용자 급수 자동 판정
- **발음 듣기**: TTS 연동으로 단어/문장 발음 재생
- **학습 알림**: 복습 주기에 맞춰 푸시 알림 발송

### 8.2 중기 확장 (6-12개월)

- **개인화 추천**: 학습 패턴 분석 기반 맞춤 기사 추천
- **작문 연습**: 학습한 단어/표현으로 문장 작성 연습
- **소셜 기능**: 친구 추가, 학습 경쟁, 리더보드

### 8.3 장기 확장 (12개월 이후)

- **다언어 확장**: 스페인어, 독일어, 이탈리아어 등 CEFR 기반 언어
- **모바일 앱**: iOS/Android 네이티브 앱 출시
- **프리미엄 기능**: 고급 분석, 무제한 저장, 광고 제거
- **AI 튜터**: 1:1 대화형 학습 지원
